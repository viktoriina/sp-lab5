# System Programming | Lab 5

> This is lex/yacc parser implementation in `C`.

Made with [bison](https://www.gnu.org/software/bison/).

## How to use?

Compile down:

```bash
make calculator
```
Execute:

```bash
./calculator
```

## Arithmetic inputs

This implementation supports arithmetic parsing.
Example input to the program:

```bash
a = log(3)/2
sin(a * 8 - 3) - 3 * 123
```
